# GIT instruction   

## Creating a **repository:**
```sh
git init
```
## Adding a modifided file into git system:
```sh
git add
```
## Adding a  commit:
```sh
git commit -m ""
```
## Observing history:
```sh
git log

git log --oneline
```
## Returning to previous versions:
```sh
git checkout

git checkout master
```
